📂 Excel VBA File Management Automation

Automatically organize mixed files into categorized folders using Excel VBA and FileSystemObject (FSO).

 🚀 Features

- Select Source Folder
- Select Destination Folder
- Automatically detect file types
- Create folders dynamically
- Organize files based on their extensions
- One-click automation

Supported File Types

| Category | Extensions |
|----------|------------|
| Excel | xls, xlsx, xlsm, xlsb |
| SQL | sql |
| PDF | pdf |
| CSV | csv |
| Word | doc, docx |
| PowerPoint | ppt, pptx |
| Images | jpg, jpeg, png, gif, bmp, webp |
| Videos | mp4, mov, avi, mkv |
| Audio | mp3, wav |
| Power BI | pbix |
| Archives | zip, rar, 7z |
| Others | Remaining file types |
